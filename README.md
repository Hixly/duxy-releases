# Duxy — Releases

> A desktop AI companion that lives on your machine. Less chatbot, more co-pilot.

Duxy follows your cursor, points out things on your screen, and talks to you. This repo hosts the Windows release artifacts.

## Download

Grab the latest installer from the [Releases page](https://github.com/Hixly/duxy-releases/releases) — current version: **v0.1.0**.

**Requirements:** Windows 10/11, 64-bit.

## Install

1. Download the installer `.exe`
2. Run it — Windows SmartScreen may warn on an unsigned build; choose **More info → Run anyway**
3. Duxy lives in your system tray after launch
